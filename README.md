# 💫 About Me:
I’m a game developer with a degree in Information Engineering for Videogames and AR. I enjoy turning ideas into playable experiences—whether it’s building systems, prototyping mechanics, or shaping the feel of a moment. Games like Dark Souls and Undertale influence the way I think about atmosphere, emotion, and player agency. I’m now looking for a team where I can grow, collaborate, and help create games that resonate with players the way my favorite games resonated with me.
